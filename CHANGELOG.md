# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-07

### Added

- Azure Cosmos DB account resource with configurable consistency policies
- SQL API (NoSQL) support with databases and containers
- MongoDB API support with databases and collections
- Multi-region geo-replication with automatic failover
- Multi-region writes support
- Analytical storage (Azure Synapse Link) integration
- Autoscale throughput for databases and containers/collections
- Private endpoint with DNS zone group support
- Customer-managed key (CMK) encryption
- Configurable backup policies (Periodic and Continuous)
- Virtual network rules and IP range filtering
- Free tier support
- Diagnostic settings for monitoring
- Basic, advanced, and complete usage examples
